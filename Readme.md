## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/fatihkan/card_game_v1.git
```

Proje dizinine gidin

```bash
  cd card_game_v1
```

Gerekli paketleri yükleyin

```bash
  npm install yada yarn install
```

Sunucuyu çalıştırın

```bash
  npm run start yada yarn start yada expo start
```
